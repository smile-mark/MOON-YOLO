MOON-YOLO
ALightweight and High-Precision Infrared Person Detection Model for Field Search and Rescue

This repository provides the re-annotated label files for the POP Dataset, specifically adapted for the MOON-YOLO infrared personnel detection model. The complete source code of MOON-YOLO will be released soon.

Copyright and Distribution of Original Imagery

The copyright of the raw infrared images from the POP dataset is not held by this project. Accordingly, this repository does not include or distribute the source imagery directory.

Contents of This Repository

The full set of re-annotated label files.
Predefined train/val/test data split configurations.

Usage and Redistribution Policy

As we do not hold the copyright to the original POP dataset imagery, redistribution or public sharing of the raw images is not permitted. This repository only provides re-annotated label files. Users must independently obtain the raw images from the official source before using these annotations.

To obtain the complete dataset, please download the original imagery as described in the official publication:

Song, Z., Yan, Y., Cao, Y., et al. An infrared dataset for partially occluded person detection in complex environment for search and rescue. Scientific Data 12, 300 (2025). https://doi.org/10.1038/s41597-025-04327-7
